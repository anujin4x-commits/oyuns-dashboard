# OYUNS Санхүүгийн бүртгэл

Утсанд тохиромжтой санхүүгийн dashboard.

## Deploy хийх заавар

### 1. GitHub repo үүсгэх
1. [github.com](https://github.com) нэвтрэх
2. **New repository** дарах
3. Repository name: `oyuns-dashboard`
4. **Public** сонгох
5. **Create repository** дарах

### 2. Файлуудыг upload хийх
```
Repo дотор `uploading an existing file` дарж бүх файлуудыг drag & drop хийнэ
```

### 3. GitHub Pages идэвхжүүлэх
1. Repo → **Settings** → **Pages**
2. Source: **GitHub Actions** сонгох
3. Хадгалах

### 4. Deploy автоматаар болно
Файл upload хийгдмэгц ~2 минутад:
`https://[таны-username].github.io/oyuns-dashboard/`

## Өгөгдөл хадгалалт
Бүх өгөгдөл утасны/компьютерийн **localStorage**-д хадгалагдана.
